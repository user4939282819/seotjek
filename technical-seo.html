Technical SEO: Teknisk Optimering Grundlæggende

Technical SEO handler om at få Google til at kunne crawle, indeksere og rank din hjemmeside. Uden teknisk basis, har du intet.

Sitehastighed (Page Speed)

  • Mobile skal være under 3 sekunder på 4G
  • Desktop ideelt under 2 sekunder
  • Komprimér billeder (TinyPNG)
  • Minimer CSS/JavaScript
  • Brug CDN for globale besøgende

Mobile-First

  • Google prioriterer mobile version
  • Responsivt design er vigtigt
  • Touch-buttons mindst 44x44px
  • Teksterne skal være læsbare uden zoom

Core Web Vitals

  • LCP: Stort element skal loade på under 2,5 sekunder
  • FID: Interaktivitet under 100ms
  • CLS: Minimalt layout shift

Crawlability

  • XML sitemap.xml (alle sider)
  • robots.txt som tillader Googlebot
  • Ingen duplicate content
  • HTTPS aktiveret (standard nu)

Struktureret Data (Schema)

  • Organization schema
  • Article schema for blogindlæg
  • LocalBusiness for fysiske butikker
  • FAQ schema for FAQ-sider

Technical SEO er øl-fondament. Hvis det ikke er på plads, intet andet virker.
