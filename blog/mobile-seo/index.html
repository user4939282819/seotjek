Mobile SEO Optimization 2024

Sådan optimerer du for mobil søgninger og får top ranking

Mobile-First Index

Google ranker nu først baseret på mobile version af din site. Hvis din mobile version er dårlig, falder du.

Core Web Vitals for Mobil

  • LCP (Largest Contentful Paint): <2.5 sekunder
  • FID (First Input Delay): <100ms
  • CLS (Cumulative Layout Shift): <0.1

Mobile Usability

  • Responsive design (ikke mobile-specific site)
  • Stor nok tekst (min 16px)
  • Klikbare buttons (min 48x48px)
  • Ingen interstitials/pop-ups

Mobile Loading Speed

Mobil internet er langsommere. Din site skal under 3 sekunder på 4G.

Mobile Keywords

Mobile søg er ofte anderledes end desktop. Mennesker søger "quick [service]" eller "near me [service]" på mobil.

AMP (Accelerated Mobile Pages)

AMP kan hjælpe med speed, men det er ikke påkrævet for god ranking.
